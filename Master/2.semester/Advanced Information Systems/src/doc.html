Postup instalace

Backend

  • java
  • Spring boot
  • funkční instalace maven
  • Vytvoření DB se jménem "pis-library"
  • Následně pomocí příkazu "mvn spring-boot:run"
Frontend
  • node.js
  • nainstalované nástroje npm a yarn
  • instalace potřebných knihoven pomocí "yarn install"
  • spuštění frontendu pomocí příkazu "yarn serve"
